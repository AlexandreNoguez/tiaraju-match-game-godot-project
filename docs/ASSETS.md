# Assets de Terceiros

## Objetivo

Registrar os assets gratuitos de terceiros usados no projeto, suas origens, licencas e a pasta onde foram organizados.

## Packs instalados

### Kenney UI Pack

- origem: https://kenney.nl/assets/ui-pack
- licenca: `CC0`
- pasta: `assets/third_party/kenney/ui-pack`
- uso sugerido:
  - botões
  - paineis
  - fontes temporarias
  - efeitos sonoros de interface

### Kenney Cursor Pack

- origem: https://kenney.nl/assets/cursor-pack
- licenca: `CC0`
- pasta: `assets/third_party/kenney/cursor-pack`
- uso sugerido:
  - cursores customizados
  - ponteiros de interacao

### Kenney Game Icons

- origem: https://kenney.nl/assets/game-icons
- licenca: `CC0`
- pasta: `assets/third_party/kenney/game-icons`
- uso sugerido:
  - icones de HUD
  - objetivos
  - status
  - menus e botoes

## Regras de uso no projeto

1. Priorizar assets `CC0` ou licencas claramente permissivas.
2. Manter todo asset de terceiro dentro de `assets/third_party/`.
3. Registrar origem e licenca neste arquivo ao adicionar novos packs.
4. Evitar misturar assets temporarios com arte final do jogo.
5. Personagens principais e identidade cultural do projeto devem ser autorais.

## O que nao versionamos

- zips baixados em `assets/third_party/kenney/downloads/`
- atalhos `.url` incluidos por alguns packs

## Observacao

Estes packs servem muito bem para prototipo, HUD e interface. Para a identidade final do jogo, o ideal e produzir arte propria.
